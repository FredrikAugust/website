LibreRead
Self-hosted Free(Libre) Ebook Reader
Keep all your ebooks in one place. Access it on any device.
Try demo Install
Features
  • - Easy installation
  • - Built using Golang
  • - Browser-based
  • - Responsive design
  • - Full-text search
  • - Highlight & Annotate
  • - Supports PDF & EPUB
